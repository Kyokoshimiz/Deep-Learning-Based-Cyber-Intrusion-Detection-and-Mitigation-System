# Deep Learning Based Cyber Intrusion Detection and Mitigation System

## Overview
This project implements a Deep Learning based system to detect and mitigate cyber intrusions by analyzing real-time threat data. The model processes network indicators such as IP addresses, domains, and URLs to classify malicious activities.

## Features
- Real-time threat data collection
- Dataset preprocessing and cleaning
- Deep learning based intrusion detection
- Threat classification
- Security alert generation

## Technologies Used
- Python
- PyTorch
- Pandas
- NumPy
- Machine Learning / Deep Learning

## Dataset
The dataset consists of threat intelligence data including:
- Malicious IP addresses
- Suspicious URLs
- Domains
- Network activity indicators

## Model
A deep learning model is used to detect patterns in network traffic and classify potential cyber attacks.
